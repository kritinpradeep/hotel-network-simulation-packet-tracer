# Hotel Network Infrastructure Simulation

## 📌 Overview
This project simulates a real-world hotel network using Cisco Packet Tracer, connecting multiple departments with secure and efficient communication.

## 🏨 Network Areas
- Reception  
- First Floor  
- Second Floor  
- Office Area  
- Security Area  
- Server Room  

## 🚀 Features
- Multi-network architecture with routers and switches  
- DHCP for automatic IP assignment  
- DNS and Email servers  
- VoIP using IP phones  
- Firewall (ASA) for network security  
- Inter-network communication  

## 🛠️ Technologies Used
- Cisco Packet Tracer  
- Networking Protocols (TCP/IP, DHCP, DNS)  
- Routing & Switching  

## 🌐 Network Design
The network is divided into multiple LANs connected through routers, ensuring communication between departments and secure internet access.
